# Cursor prompts

Copy-paste these into Cursor chat. No terminal knowledge required.

## One-time setup

```
Set up my podcast — my show is called [SHOW NAME], host is [HOST NAME].
Help me edit podcast.config.yaml with a good title template and default keywords.
```

```
Where do I put my RSS.com API key and podcast ID? Walk me through copying .env.example to .env.
```

```
Run make check and tell me if anything needs fixing.
```

## New episode

```
Create a new episode about [TOPIC].
```

```
New episode: interview with [GUEST] about [TOPIC].
```

```
Start a sports preview episode for [TEAM A] vs [TEAM B] in [COMPETITION].
```

## Writing content

```
Turn my script into show notes for description.md. Keep it under 4000 characters and use the style in templates/descriptions/generic-description.md.
```

```
Expand my bullet notes into a full script.md using templates/scripts/generic-script.md.
```

## Before publishing

```
Dry run episodes/[FOLDER] and tell me if anything is wrong.
```

```
Check that my MP3, description, and episode.yaml are ready to publish.
```

## Publishing

```
Publish this episode — run dry-run first, then publish if it looks good.
```

```
Run make publish for episodes/[FOLDER] and show me the dashboard link.
```

## Transcript fixes

```
Add these transcript corrections to corrections.yaml: "[wrong]" should be "[right]".
```

```
Re-apply transcript corrections for RSS.com episode ID [ID].
```
